Add `@bunny.net/database-openapi` package

Generates an OpenAPI 3.0.3 specification from a `DatabaseSchema` object. Zero
dependencies — pass in a schema, get back a spec with CRUD paths, typed schemas
(base, insert, update), error responses, reusable parameters, top-level tags,
and name-aware example values for common column names.
